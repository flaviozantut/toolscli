## toolscli - Laravel tolls for comand line





## Installation

Installation with Laravel Artisan

	php artisan bundle:install toolscli

## Bundle Registration

	'toolscli',

## Usage

	php artisan toolscli::build [task] [parameters]

use -c to compress css file
